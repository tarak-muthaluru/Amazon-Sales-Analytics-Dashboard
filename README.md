## 🛒 Amazon Sales Data Analysis & Dashboard

## 📌 Overview

This project analyzes Amazon sales data to uncover key business insights and visualize them using an interactive Power BI dashboard. The goal is to understand sales trends, customer behavior, and product performance.



## 🎯 Objectives

* Clean and preprocess raw Amazon dataset
* Perform exploratory data analysis (EDA)
* Identify key sales patterns and trends
* Build an interactive dashboard for decision-making



## 📂 Project Files

* `Amazon.csv` → Dataset used for analysis
* `Amazon Dash Board.pbix` → Power BI dashboard file
* `Amazon Dash Board 1.png` → Dashboard preview
* `Amazon Dash Board 2.png` → Dashboard preview



## 🛠️ Tools & Technologies

* Python (Data Cleaning & Analysis)
* Power BI (Dashboard Visualization)
* Pandas, NumPy (Data Processing)



## 🔍 Key Features

* Data cleaning and preprocessing
* Interactive dashboard with filters and visuals
* Sales trend analysis
* Product/category performance insights



## 📊 Dashboard Insights

* Identified top-performing products and categories
* Analyzed sales distribution across different segments
* Observed trends and patterns in customer purchasing behavior
* Highlighted key metrics for business decision-making


## ⚙️ How to Use

1. Download the repository
2. Open `Amazon Dash Board.pbix` using Power BI Desktop
3. Explore the dashboard with filters and visuals



## 📈 Dataset Details

* Source: Kaggle
* Records: ~100,000 entries
* Type: Structured sales data
